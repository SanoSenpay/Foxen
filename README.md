✨ **О РАСШИРЕНИИ**

**Foxen** — полностью бесплатное браузерное расширение с открытым исходным кодом для продавцов FunPay.

Foxen добавляет AI-инструменты, глубокую кастомизацию интерфейса, автоматизацию рутинных задач, аналитику продаж, автоответчик, авто-выдачу и десятки других функций — всё, чтобы зарабатывать больше, тратя меньше времени.

> **Firefox Edition** — эксклюзивные функции: полный экспорт данных и расширенная настройка звуков.
